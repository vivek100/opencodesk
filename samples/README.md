# Sample datasets

Realistic files for uploading in OpenCoDesk and testing agent analysis workflows.

## Files

| File | Rows | Best for |
|------|------|----------|
| `superstore-sales.xlsx` | 2,000 | Regional revenue, profit margins, product/category analysis (sample) |
| `superstore-sales.csv` | ~9,994 | Full dataset as CSV |
| `sales-q1.xlsx` | 16 | Quick smoke tests — executive summary, revenue by region |
| `sales-q1.csv` | 16 | Same data as CSV |
| `retail-inventory.xlsx` | 48 | Stock levels, reorder alerts, anomaly / data quality checks |
| `expense-claims.xlsx` | 48 | Department spend, approval status, missing-receipt edge cases |

## Suggested prompts

**Superstore (full dataset)**
> Upload superstore-sales.xlsx and create an executive summary with top regions, categories, and profit drivers. Save the report to outputs and show it.

**Sales Q1 (quick test)**
> Analyze sales-q1.xlsx and calculate revenue by region and product. Save a CSV summary to outputs and show it.

**Inventory**
> Review retail-inventory.xlsx for items below reorder point by region. Save findings to outputs.

**Expense claims**
> Find expense claims with missing receipts or pending status. Save a cleaned summary to outputs.
